于是乎一整天什么都没干，就折腾这玩意儿了！！！

升级Butterfly

关于该主题的安装以及升级，主题作者说的很清楚，这里就不赘述了，具体可查看这里：Butterfly主题安装和升级。

git pull

  • Butterfly使用了data file特性，可直接使用git pull直接升级
  • 之后运行hexo-clean && hexo-g && hexo-s,查看结果
